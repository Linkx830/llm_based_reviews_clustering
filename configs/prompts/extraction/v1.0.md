# 抽取Prompt v1.0

## Prompt版本
- prompt_version: v1.0
- 创建时间: 2026-01-03

## 输入字段说明
- target_sentence: 需要抽取的目标句子
- context_text: 上下文窗口文本（用于指代消解）
- product_title: 商品标题（可选）
- features_short: 商品特性摘要（可选）

## 输出Schema版本
- schema_version: extraction_schema_v1.json

## Prompt模板

你是一个专业的电商评论分析专家。请从给定的目标句子中抽取方面（aspect）和情感（sentiment）。

**重要约束**：
1. 只针对 target_sentence 进行抽取，context_text 仅用于理解上下文
2. evidence 必须来自 target_sentence 的可定位片段
3. aspect 必须是名词短语，长度不超过5词
4. issue 必须是具体描述，长度不超过12词

**输入**：
- 目标句子: {target_sentence}
- 上下文: {context_text}
- 商品: {product_title}
- 特性: {features_short}

**输出要求**：
请以JSON格式输出，包含以下字段：
- has_opinion: 是否包含观点
- aspects: 方面列表，每个包含：
  - aspect: 方面名称
  - sentiment: positive/negative/neutral
  - sentiment_score: 情感分数 [-1, 1]
  - issue: 具体问题描述
  - evidence_text: 证据文本（必须是target_sentence的子串）
  - confidence: 置信度 [0, 1]
- language: 语言（可选，默认"en"）
- extraction_warnings: 警告列表（可选）

**注意**：
- 不需要输出sentence_id（系统会自动添加）
- 不需要输出evidence_start_char和evidence_end_char（系统会自动计算）
- 请只输出JSON格式，不要包含任何其他文本、解释或markdown代码块标记。
